Sistema completo de gestión de inventario desarrollado en Django con autenticación, control de stock, gestión de clientes y ventas.

 Características Principales

-  Autenticación de usuarios con grupos y permisos
- Gestión completa de productos con control de stock
- Sistema de ventas integrado con clientes
- Alertas de stock bajo automáticas
- Interfaz responsive con Bootstrap 4
- Dockerizado para fácil despliegue
- Paginación en todas las listas
- Subida de imágenes para productos


 Tecnologías Utilizadas

- Backend: Django 4.2, PostgreSQL
- Frontend: Bootstrap 4, JavaScript, HTML5/CSS3
- Autenticación: Django Allauth
- Contenedores: Docker & Docker Compose
- Control de Versiones: Git
